# Admin panel

This is the admin panel for the assessment

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository.git
   
2. Install dependencies:
  ```bash
  npm install
```

3. Start the development server:
  ```bash
  npm run start
```
